# Google OAuth - Frontend Quick Reference

## What Frontend Needs to Do

### 1. Add Google Sign-In Button
```html
<button (click)="signInWithGoogle()">Sign in with Google</button>
```

### 2. Redirect to Backend Endpoint
```typescript
signInWithGoogle() {
  window.location.href = 'http://your-api-url/api/auth/google';
}
```

### 3. Handle Callback Route
Create route: `/auth/callback`

### 4. Extract Token from Callback
```typescript
// In your callback component
this.route.queryParams.subscribe(params => {
  if (params['token']) {
    localStorage.setItem('token', params['token']);
    // Redirect to dashboard
  } else if (params['error']) {
    // Show error message
  }
});
```

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/auth/google` | Start Google OAuth (redirects to Google) |
| GET | `/api/auth/google/callback` | Google callback (handled by backend, redirects to frontend) |

## Callback URL Format

**Success:**
```
/auth/callback?token={JWT}&email={EMAIL}&firstName={NAME}&lastName={NAME}&role={ROLE}
```

**Error:**
```
/auth/callback?error={ERROR_MESSAGE}
```

## Complete Example

```typescript
// auth.service.ts
signInWithGoogle() {
  window.location.href = `${API_URL}/api/auth/google`;
}

// auth-callback.component.ts
ngOnInit() {
  this.route.queryParams.subscribe(params => {
    if (params['token']) {
      localStorage.setItem('token', params['token']);
      this.router.navigate(['/dashboard']);
    } else if (params['error']) {
      this.router.navigate(['/login'], { 
        queryParams: { error: params['error'] } 
      });
    }
  });
}
```

## Notes

- Token is JWT format (same as email/password login)
- Store token in localStorage
- Include token in Authorization header: `Bearer {token}`
- Users can have both Google and Email auth (auto-linked by email)
